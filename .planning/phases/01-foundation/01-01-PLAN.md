# Plan 01-01: Repo & Module Init

## Goal
Create the Go module, directory tree, Makefile, linter config, and placeholder main files so `go build ./...` succeeds.

<threat_model>
## Threat Model

**Scope:** Project skeleton, build tooling, directory layout. No runtime network code in this plan.

| Threat | Severity | Mitigation |
|--------|----------|------------|
| Malicious dependency injection via go.mod | Medium | No third-party dependencies added in this plan; go.mod only declares the module path and Go version |
| Secrets committed to repo | Low | `.gitignore` includes `bin/` and binary patterns; no `.env` or credential files created |
| Build artifact pollution | Low | `make clean` target removes `bin/`; `.gitignore` prevents committing binaries |

**Verdict:** No high-severity threats. Proceed.
</threat_model>

## must_haves
- `go.mod` exists with module `github.com/fwardzic/mcast-test-app` and go 1.24
- Directory tree: `cmd/sender/`, `cmd/receiver/`, `internal/{config,packet,multicast,display}/`, `k8s/`
- `Makefile` has targets `build`, `test`, `lint`
- `.golangci.yml` exists
- `go build ./...` exits 0
- `.gitignore` includes Go binary patterns

<task id="01-01-3">
<title>Create Makefile</title>
<read_first>
- Makefile (does not exist yet)
</read_first>
<action>
Create `Makefile` with these exact targets:

```makefile
.PHONY: build test lint clean

build:
	go build -o bin/sender ./cmd/sender
	go build -o bin/receiver ./cmd/receiver

test:
	go test -v -race ./...

lint:
	golangci-lint run ./...

clean:
	rm -rf bin/
```
</action>
<acceptance_criteria>
- `Makefile` contains `go build -o bin/sender ./cmd/sender`
- `Makefile` contains `go build -o bin/receiver ./cmd/receiver`

- `Makefile` contains `golangci-lint run ./...`
- `make build` exits 0
</acceptance_criteria>
